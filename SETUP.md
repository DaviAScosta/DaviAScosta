# Como publicar o perfil

## 1. Enviar o README

```powershell
cd C:\Users\davia\Projects\DaviAScosta
git add README.md
git commit -m "feat: perfil focado em skills"
git push
```

Se ainda não inicializou:

```powershell
git init
git add .
git commit -m "feat: README de perfil"
git branch -M main
git remote add origin https://github.com/DaviAScosta/DaviAScosta.git
git push -u origin main
```
